# Mondo
Some themes for SLiM login manager



This is a simple them of SLiM login manager, teste under Slackware 14.2
Requirement: slim program installe on your machine.
Install:
1 - get root access
2 - make directory /usr/share/slim/themes/mondo
3 - copy all files under mondo directory
4 - copy slim.conf under /etc  directory
